Лабораторная работа 3. Классы. Основные принципы ООП. Полиморфизм, наследование , инкапсуляция.
Задание 1. 
Описать класс A, в котором должно содержаться: минимум два конструктора с различными аргументами, деструктор, минимум два приватных поля произвольного типа, для каждого из полей публичные геттер и сеттер, метод, вызов которого будет выводить на экран “I am a base class method ”.
Описать класс B, который является наследником класса A. В этом классе должно содержаться: минимум один конструктор, деструктор, минимум одно приватное поле произвольного типа, для каждого поля публичные геттер и сеттер, перегруженный метод родительского класса который будет выводить на экран “I am a derived class method”.
В конструкторы и деструкторы этих классов необходимо добавить отладочный вывод, например: “Constructing class A…”.
Далее, создать 3 функции (НЕ МЕТОДЫ!), в которых написать так:
1:
    A *a = new A();
    a->foo();
    delete a;
2:
	B *b = new B();
	b->foo();
	delete b;
3:
	A *a = new B();
	a->foo();
	delete a;
    
И вызвать их поочередно в main(). При защите необходимо знать что такое полиморфизм, инкапсуляция и наследование, а также любое ключевое слово, использованное вами. В лабораторной работе не должно быть лишнего кода. Пример вызова третьей функции:

Constructing class A...
Constructing class B...
I am a derived class method
Destructing class B...
Desctructing class A...

Названия классов и методов произвольные.  Суть работы в понимании студентом принципов ООП и как ими можно воспользоваться.
